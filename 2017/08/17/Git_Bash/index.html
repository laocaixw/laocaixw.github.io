Git 的一些命令

git init

将当前目录初始化为一个 git 仓库。

git status

查看当前状态。

git add

提交缓存。将修改添加到提交缓存中，只有 git commit 之后才算真正提交了。

git commit

提交。把 git add 好的代码提交待仓库。

git log

查看提交日志。就是 commit 的记录。

git reset

撤销commit

git branch

分支。

合并分支后，可以删除分支：

git checkout

切换分支。创建分支后，当前所在分支还是之前的分支。

远程库分支相关

git merge

合并分支。

git tag

标签。可以给代码库打标签，如版本号。

如果需要查看之前版本（标签）的代码，可以通过 git checkout v1.0 切换回原来标签位置的代码状态。

git stash

在开发中会遇到开发一半，需要修复bug的情况，这时候就可以用stash命令将开发到一半的代码暂存起来。

git fetch
